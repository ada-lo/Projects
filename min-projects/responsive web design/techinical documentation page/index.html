Introduction

C is a powerful general-purpose programming language. It is fast, portable, and available on all platforms. It was developed in the early 1970s by Dennis Ritchie and remains widely used today for system and application development.

C is commonly used to develop operating systems, embedded systems, databases, and more.

int main() { printf("Hello, World!"); return 0; }
Syntax

C programs follow a simple structure:

  • Preprocessor commands (e.g., #include <stdio.h>)
  • Function declarations and definitions
  • Statements within blocks of code (using braces {})

A basic C program that prints "Hello, World!" looks like this:

#include <stdio.h> int main() { printf("Hello, World!"); return 0; }
Data Types

C supports several built-in data types:

  • int: Integer data type
  • float: Floating point number
  • char: Character type
  • double: Double precision floating point number

For example, declaring variables of different data types:

int age = 30; float pi = 3.14; char initial = 'A';
Control Structures

C includes control structures for decision making and loops:

  • if and else for decision-making
  • for loop for iteration
  • while loop for conditional looping

An example of an if condition in C:

if (age > 18) { printf("You are an adult."); }

Example of a for loop in C:

for (int i = 0; i < 10; i++) { printf("%d ", i); }
Functions

Functions in C allow you to break the program into reusable blocks of code. A function is defined with a return type, a name, and parameters.

The basic syntax of a function in C:

return_type function_name(parameters) { // function body }

For example, a function that adds two numbers:

int add(int a, int b) { return a + b; }

To call the function:

int result = add(5, 3);
